Diferenças entre id e class!

Um id é único, a class pode se repetir.

Na folha de estilos quando eu configuro um id, eu utilizo #.

Aqui estouu usando uma class. Eu posso selecionar vários elementos utilizando uma class.

Aqui estou repetindo o uso da class criada anteriromente: "conteudo"

Na folha de estilos quando eu configuro uma class, eu utilizo .
